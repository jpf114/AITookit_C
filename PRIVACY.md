# AIToolkit 隐私政策

**生效日期：** 2026-05-23  
**适用产品：** AI 检测工具（AIToolkit）v1.0.1 及后续版本

## 概述

AIToolkit 是一款本地运行的桌面应用程序。我们重视您的隐私。本政策说明本应用如何处理数据。

## 数据处理原则

**所有 AI 推理均在您的设备本地完成。** 您的图像、视频及推理结果不会上传至我们的服务器。

## 我们收集的数据

### 本地存储（不上传）

| 数据类型 | 存储位置 | 用途 |
|----------|----------|------|
| 应用设置 | Windows 注册表（QSettings） | 语言、GPU 选项、线程数、窗口120目录等 |
| 最近使用的模型/图像路径 | 同上 | 快捷访问 |
| 窗口位置与大小 | 同上 | 恢复界面布局 |
| 崩溃转储文件 | `%LOCALAPPDATA%\AIToolkit\crash_dumps\` | 本地调试（可能含内存快照） |

### 网络活动

本应用 **不包含遥测或分析功能**。仅在以下情况下发起网络连接：

- **模型下载**：当您主动点击「下载示例模型」或「模型目录」时，从 GitHub（Ultralytics 官方发布页）下载 ONNX 模型文件
- **检查更新**（若启用）：查询 GitHub Releases API 获取最新版本号

我们不会通过这些连接收集个人身份信息。

## 我们不收集的内容

- 无用户账户或登录
- 无使用行为追踪
- 无广告
- 无第三方分析 SDK

## 崩溃报告

应用崩溃时会在本地生成 MiniDump 文件（`.dmp`），并弹出提示告知保存路径。崩溃报告 **不会自动上传**。您可自行决定是否将 dump 文件发送给开发者以协助排查问题。

## 第三方模型许可

通过模型目录下载的 Ultralytics YOLO 模型采用 **AGPL-3.0** 许可证。商业使用需遵守 Ultralytics 许可条款，详见 [ultralytics.com/license](https://ultralytics.com/license)。

## 数据安全

- 推理输入/输出由您自行管理，存储在您选择的本地路径
- 导出文件名经过 sanitize 处理，防止路径注入
- 建议仅从官方渠道下载应用安装包

## 您的权利

由于所有数据均存储在本地，您可以随时：

- 通过 Windows「应用和功能」卸载本应用
- 删除 `%LOCALAPPDATA%\AIToolkit\` 下的设置与崩溃转储
- 在设置中关闭「检查更新」

## 儿童隐私

本应用不面向 13 岁以下儿童，也不会故意收集儿童信息。

## 政策变更

重大变更将在 GitHub 仓库的 CHANGELOG 中公布。继续使用即表示接受更新后的政策。

## 联系我们

如有隐私相关问题，请通过 GitHub Issues 联系项目维护者。
