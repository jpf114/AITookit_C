## 子功能清单

### 目标检测

- [YOLO 目标检测](./目标检测/YOLO目标检测.md) — 基于 YOLO 系列模型的目标检测全流程
- [NMS 非极大值抑制](./目标检测/NMS非极大值抑制.md) — 后处理去重算法

### 图像分类

- 分类模型基于 ClassificationModel 实现，预处理为 BGR→RGB + Resize + 归一化，后处理为 softmax + Top-K 排序
- 模型清单中 `task_type` 设为 `classification`，详见 [模型清单规范](./模型配置/模型清单规范.md)

### 实例分割

- 分割模型基于 SegmentationModel 实现，后处理在检测基础上增加掩码生成
- 模型清单中 `task_type` 设为 `segmentation`，详见 [模型清单规范](./模型配置/模型清单规范.md)
